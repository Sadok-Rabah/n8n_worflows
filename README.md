# n8n_worflows
Here are my n8n projects, you'll find every workflow's description down here

1/ Telegram AI Meal Analyzer : 
Description :
This workflow is an intelligent nutritional assistant that allows a user to send a photo of their meal via Telegram and instantly receive a complete nutritional analysis. Thanks to Google Gemini Vision, the workflow identifies the foods in the image and calculates estimated values for calories, proteins, and carbohydrates. The result is then automatically sent back in the same Telegram chat, providing a smooth and interactive experience.
This type of workflow is particularly useful for personal nutrition tracking, nutritional coaching, fitness applications, or any project requiring a quick estimation of nutritional values from photos.

2/ Telegram AI Payment Logger : 
Description :
This workflow is an intelligent payment management assistant that allows a user to send payment information via Telegram. The Telegram bot acts as an interface between the user and an artificial intelligence agent that analyzes the received message to identify essential information such as the payment date, the amount, and the name of the person involved.
Once the data is extracted, the workflow automatically records this information in a payment notebook (Google Sheets). If some information is missing or unclear, the agent can request additional details from the user before completing the entry. A confirmation message is then automatically sent back in the same Telegram chat.
This type of workflow is particularly useful for simple and fast payment tracking, personal transaction management, or any project requiring automated payment recording through a conversational interface.
