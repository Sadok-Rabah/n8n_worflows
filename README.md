1/ Telegram AI Meal Analyzer :
Description :

Ce workflow est un assistant nutritionnel intelligent qui permet à un utilisateur d’envoyer une photo de son repas via Telegram et de recevoir instantanément une analyse nutritionnelle complète.
Grâce à Google Gemini Vision, le workflow identifie les aliments sur l’image et calcule des valeurs estimées pour calories, protéines et glucides. Le résultat est ensuite renvoyé automatiquement dans le même chat Telegram, offrant une expérience fluide et interactive.

Ce type de workflow est particulièrement utile pour : le suivi personnel de l’alimentation, le coaching nutritionnel, les applications de fitness, ou tout projet nécessitant une estimation rapide des valeurs nutritionnelles à partir de photos.
